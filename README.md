# 亏损计算系统

跨境电商利润计算工具，支持 Wildberries店铺利润计算、佣金匹配、折扣推算等功能。

## 功能特性

- **利润计算**: 支持跨境电商多店铺（本地/跨境）模式下的利润计算
- **佣金匹配**: 两步匹配（SKU → 商品 → 品类）自动获取佣金
- **折扣推算**: 基于利润目标的折扣推荐
- **数据导入**: 支持 Excel 模板导入
- **数据库管理**: 内置 SQLite 数据库存储

## 导出策略

系统支持 3 种导出策略，通过设置抽屉的「导出策略」下拉框选择：

| 策略 | 折扣列(L) | 价格列(J) | 适用场景 |
|------|----------|----------|---------|
| **折扣优先** | 写入目标折扣(≤0时清零) | 不改 | 只需调折扣，不调价格 |
| **保持折扣调价** | 不改 | 写入新原价(target_new_price) | 维持当前折扣率，抬高原始定价来达标 |
| **折扣归零调价** | 清零为0 | 写入目标售价(target_price) | 不打折，实价实标 |

**导出筛选**：在设置抽屉的「导出筛选」分组中，可按「利润值」或「利润率」设置阈值，仅导出低于该阈值的商品。

**价格取整**：导出时所有价格自动向上取整到整数（如 601.29 → 602）。

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
python src/main.py
```

或直接运行打包好的 exe：

```bash
dist/WB系统.exe
```

## 技术栈

- Python 3.10+
- PyQt5 / PySide6 (GUI)
- SQLite (数据库)
