## v2.3.1 (2026-04-26)

### 修复问题

#### 飞书模块 v1.3.0
- **修复同步数据重叠** — 每次同步前清空旧产品数据，保证数据库与飞书表格完全一致（无重复/无遗漏）
- **修复旧数据库缺少唯一约束** — 自动检测并重建 products 表 UNIQUE(sku_code, sheet_name) 约束，数据迁移不丢失
- **图片增量下载优化** — 基于数据库中已存储的 fileToken 对比，相同图片不重复下载；下载后自动更新 image_local_path
- **"Data not ready" 递增等待** — 飞书服务端数据未就绪时采用递增等待重试（3s→6s→9s），替代固定1秒等待
- **批量请求列范围优化** — 使用实际列数替代 ZZ(702列)，数据量减少 15-30 倍，避免 504 超时
- **降低并发避免文档阻塞** — 降级逐个读取时线程数从 4 降至 2，符合飞书官方"避免并发读取同一文档"的建议
- **连续失败阈值提高** — 从 5 次提高到 20 次，避免多 Sheet 并发时互相连带中止
- **同步失败提示改进** — 同步 0 条数据时弹窗标题改为"同步失败"，附可能原因和解决建议

## v2.2.1 (2026-04-24)

### 性能优化

#### 数据库性能
- **新增索引** — products(sku_code)、products(sheet_name)、commissions(category)、commissions(product)，查询速度大幅提升
- **executemany 批量插入** — 产品/佣金批量同步使用 `executemany` 替代逐行循环，批量写入性能提升约 10 倍
- **SQLite PRAGMA 优化** — `synchronous=NORMAL`、`cache_size=64MB`、`temp_store=MEMORY`，减少磁盘 I/O

#### 内存优化
- **表格内存精简** — ProductViewer 每个单元格仅存储 `{"id": ...}` 替代完整数据字典，内存占用降低约 97%
- **QColor 缓存** — 库存状态高亮颜色使用类变量缓存，避免重复解析 hex 字符串

#### UI 渲染优化
- **resizeEvent 防抖** — SyncWidget 布局刷新仅在窗口宽度变化 ≥100px 时触发，消除拖拽时的连续重建

### 修复问题

#### 飞书模块 v1.1.1
- **同步失败停止机制** — 连续 5 次 API 请求失败后自动停止同步，不再无限重试；通过 `SyncAbortException` 立即中断所有线程
- **取消同步按钮** — 同步进行中按钮变为红色"取消同步"，点击可中断同步线程
- **修复同步卡死** — `_populate_table` 填充期间阻塞 `cellChanged` 信号，防止触发 `_on_cell_changed` → `_filtered_data()` 导致 O(n²) 卡死

#### 产品数据查看器
- **移除表头漏斗箭头** — 去掉自定义 FilterHeader 绘制的筛选图标，改用标准 QHeaderView + 右键表头弹出筛选菜单
- **列宽自适应内容** — 使用 `resizeColumnsToContents()` 替代固定列宽映射，所有列（含 ID 列）自动适配内容宽度
- **修复清除筛选按钮无效** — `setSortIndicator(-1)` 在未启用排序时抛异常导致方法中断
- **库存状态高亮** — 库存信息列根据状态值（货源充足/停止上架/货源紧缺/等待补货）显示不同颜色

### 代码清理
- 删除未使用的 `DataViewerWidget` 兼容类
- 删除未使用的 `UploadWidget`（整个文件）
- 删除 `sync_widget.py` 中的 `DataCard` 和 `ApiSettingsDialog` 兼容包装器
- 删除 `feishu_service.py` 中的重复 logger 声明

## v1.0.0 (初始版本)

### 核心功能
- **折扣推算** — 读取 Excel 模板，批量计算盈亏，支持折扣策略/价格策略/组合策略
- **跨境/本土模式** — 跨境店铺 CNY 计价，本土店铺 RUB 计价
- **飞书数据同步** — 从飞书云表格同步产品数据(货号、分销价、尺寸、重量等)
- **佣金管理** — 品类级/商品级佣金率配置
- **汇率同步** — 自动获取 CNY→RUB 实时汇率
- **成本参数配置** — 平台佣金率、物流费、包装费、广告成本等
- **结果导出** — 将计算结果写回 Excel，仅更新折扣/价格列，保持原始格式
- **精确匹配** — 货号严格匹配，分销价格=成本价格

### 界面设计
- Material 3 + VS Code Modern 风格
- 侧边栏导航 (数据接口 / 产品数据 / 佣金数据 / 折扣推算 / 参数设置)
- 顶栏信息展示
- 状态栏进度显示

### 技术栈
- Python 3.14.2
- PySide6 6.11.0
- SQLite 数据库
- 多线程并行同步
